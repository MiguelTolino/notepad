# Notepad

We built a simply notepad where we can change text format, it includes several types of fonts and most common text styles.

## Pre-requisites

 **JVM**

## Use

  ***Just run notepad.jar***
  

## File Features

  - New file
  - Open file
  - Save file
  
## Format
  
  - **Fonts**
    - Verdana
    - Courier
    - Times New Roman
    - Arial
    
    
  - **Styles**
    - Bold
    - Italic
